# Vargas Construction Website

Professional construction company website featuring services, portfolio, and contact information.

## Features

- Dynamic video hero background
- Responsive design
- Service showcase with real project photos
- Portfolio gallery
- Contact form
- Mobile-friendly navigation

## Deployment to Netlify

### Quick Deploy

1. Push this repository to GitHub
2. Log in to [Netlify](https://www.netlify.com/)
3. Click "Add new site" → "Import an existing project"
4. Connect to your GitHub repository
5. Configure build settings:
   - **Build command:** (leave empty)
   - **Publish directory:** `/` (root directory)
6. Click "Deploy site"

Your site will be live in minutes!

### Custom Domain (Optional)

After deployment, you can add a custom domain:
1. Go to Site settings → Domain management
2. Click "Add custom domain"
3. Follow the instructions to configure your DNS

## Customization

To update contact information, edit the following in `index.html`:

- **Phone number:** Search for `(555) 123-4567` and replace with your actual number
- **Email:** Search for `info@vargasconstruction.com` and replace with your email
- **Address:** Update the location information in the contact section
- **Company info:** Update "Since 1995" if needed

## Local Development

To view locally, simply open `index.html` in your web browser.
